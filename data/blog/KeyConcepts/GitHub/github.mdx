GitHub es una plataforma de desarrollo de software confiable y segura para que los desarrolladores colaboren, codifiquen, construyan, prueben e implementen software. GitHub es utilizado por más de 100 millones de personas en todo el mundo para descubrir, bifurcar y contribuir a más de 420 millones de proyectos. Algunas de las herramientas que ofrece GitHub son:

- **GitHub Actions**: automatiza su flujo de trabajo de compilación, prueba y implementación con CI / CD simple y seguro.
- **GitHub Codespaces**: ofrece un entorno de desarrollo completo en segundos. Codifique, construya, pruebe y abra solicitudes de extracción desde cualquier repositorio.
- **GitHub Mobile**: se adapta a sus proyectos en su bolsillo, para que nunca pierda el ritmo mientras está en movimiento.
- **GitHub Codescanning**: es nuestra herramienta de análisis de código que lo ayuda a remediar problemas en su código.
- **GitHub Discussions**: crea espacio para hacer preguntas y tener conversaciones abiertas.
- **GitHub Sponsors**: le permite apoyar a sus mantenedores y proyectos de código abierto favoritos.
